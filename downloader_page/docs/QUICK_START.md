# Downloader Page — Quick Start

This folder is the **marketing / installer download site**, not the main AI Orchestrator app.

## Run locally

```bash
cd downloader_page
python start_frontend.py
```

Or:

```bash
npm install
npm run dev
```

Open http://localhost:5173
